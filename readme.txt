This plugin creates a pop up window with email form. Referred link is embedded in message area and can not be modified by sender. 

== Description ==

This Plugins creates pop up window with email form, containing three fields: sender's name, sender's email, friend's email and message content, that is referred page's URL. Message content and title of the form and button's text can be set via admin panel, where admin can set email title as well. Message content can not be modified by sender, because it is set as read only mode. Plugin can be installed via widgets admin panel, or via shortcode - [af_tf_form] placed at whatever page.

== Installation ==
1. Upload 'af-tell-a-friend' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

or upload af-tell-a-friend.zip via Plugins->Add New->Install Plugins panel and install after the file is sucessfully uploaded.

== Automatic updating of existing plugin via wordpress.org ==
1. Update
2. Deactivate
3. Activate
4. Set new options in admin Settings menu
